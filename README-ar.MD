# بروتوكول الهندسة الرئيسي (Master Engineering Protocol v8.0) 🧠

## حوّل أي ذكاء اصطناعي (AI) — من الأضعف للأقوى — إلى مهندس صارم وصادق.

## 🤔 المشكلة

إذا كنت تعتمد على الذكاء الاصطناعي في البرمجة كـ **Vibe Coder**، فمساعدك يقوم بالآتي:

| # | المشكلة | ما يحدث فعلياً |
|---|---------|--------------|
| 1 | **الهلوسة** | يكتب كود يستدعي دوال أو واجهات لا وجود لها |
| 2 | **المجاملة المفرطة** | يوافقك على كل شيء، حتى لو كانت فكرتك خطأ ومضرة |
| 3 | **غياب الفهم للمشروع** | يبدأ في كتابة الكود فوراً دون قراءة ملفات مشروعك أو قاعدة البيانات |
| 4 | **تجاهل الأخطاء الحالية** | يرى ثغرات أمنية وأخطاء كارثية في كودك ولا يخبرك بها |
| 5 | **خلط الأنماط** | يُدخل أساليب برمجة جديدة على مشروع له أسلوب ثابت (Patterns) |
| 6 | **انعدام التحذير** | يحذف ملفات أو يغير قاعدة البيانات دون أخذ إذنك |
| 7 | **تضخم النطاق (Scope Creep)** | تطلب منه تعديل زر.. فيعيد كتابة الموديول بالكامل |
| 8 | **كود غير مكتمل** | يضع تعليقات `// TODO` أو `// باقي الكود هنا` بدلاً من كود يعمل |
| 9 | **التعقيد المفرط (Over-Engineering)** | يبني أنظمة معقدة لمهمة بسيطة جداً |
| 10 | **حزم غير ضرورية** | يقترح تثبيت مكتبات `npm` مع وجود حل مدمج وجاهز |
| ... | **إلى 58 مشكلة كارثية أخرى يحلها هذا البروتوكول.** |

## 💡 الحل: دمج بروتوكول MEP مع Antigravity Kit

دمجنا الصرامة الهندسية الخاصة بـ (MEP) مع مسارات العمل الاستباقية لـ (Antigravity Kit).

```text
📁 مشروعك /
└── .agent/              ← كل شيء هنا! (مجلد الروت نظيف تماماً)
    ├── RULES.MD         ← البروتوكول الرئيسي (أشر له هكذا: @.agent/RULES.MD)
    ├── memory/
    │   ├── PROJECT_CONTEXT.MD  ← الذاكرة طويلة المدى
    │   └── MISTAKES.MD         ← سجل أخطاء الذكاء الاصطناعي
    ├── agents/          ← المتخصصون (Backend, Security, الخ)
    ├── workflows/       ← أدوات مساعدة /create, /debug
    ├── rules/           
    │   └── GEMINI.md    ← مُشغّل تلقائي لـ Cursor/Windsurf
    └── scripts/         ← فحص الجودة والأمان (Python)
```

### أفضل ما في العالمين
- فريق الـ **Antigravity Kit** يتولى "مَن" سينفذ المهمة (توجيه للمهندس المختص).
- قواعد **MEP v8.0** تتولى "كيف" ستُنفذ المهمة (إجبار الذكاء الاصطناعي على قراءة الملفات، فحص الكود، الرفض عند اللزوم).

### 🔥 كيف يعمل بروتوكولنا؟

| أداتك المفضلة | ماذا يحدث |
|-------------|-------------|
| **Cursor / Windsurf** | يتم تحميل القواعد تلقائياً عبر `always_on`. بدون أي تدخل منك. |
| **ChatGPT / Claude / Gemini** | اكتب `@.agent/RULES.MD` مرة واحدة فقط في بداية طلبك، وتعمل المنظومة بأكملها. |

**مجلد واحد. إشارة واحدة. صفر هلوّسة. حل جذري لـ 58 مشكلة.**

## 🚀 كيفية الاستخدام

### التركيب (Installation)
1. حمّل المجلد والمحتويات وضعه في الروت الخاص بمشروعك (تكون نسخة `.agent` في المجلد الأساسي).
2. لا يوجد أي إعدادات إضافية مطلوبة.

### الاستخدام الفعلي (Usage)

**Cursor / Windsurf (يعمل تلقائياً، دون قلق):**
```text
قم بإصلاح زر تسجيل الدخول لو سمحت.
```

**في ChatGPT / Claude، أو غيرها (إشارة واحدة فقط):**
```text
@.agent/RULES.MD قم بإصلاح زر تسجيل الدخول لو سمحت.
```

## 🔑 القوانين الـ 15 الحديدية للذكاء الاصطناعي

1. لا تبدأ بالكود — ابدأ بالفهم 🔴
2. لا تخمن — افحص واقرأ 🔴
3. لا تكن لطيفاً — كن هندسياً وصادقاً 🔴
4. لا تهلوس الكود — أثبت عمله أو التزم الصمت 🔴
5. لا تتجاهل المشاكل — بلّغ عنها فور دراستك للملفات 🔴
6. لا تقم بأفعال مدمرة دون أن تسأل 🔴
7. لا تتجاوز النطاق المطلوب منك 🔴
8. لا تسلم كوداً ناقصاً أو يحتوي TODOs 🔴
9. لا تكسر كوداً يعتمد عليه كود آخر دون تنبيه 🔴
10. لا تفترض — اسألني 🔴
